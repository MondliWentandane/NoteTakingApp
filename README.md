## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- React Native development environment

### Installation

1. Clone the repository:
```bash
git clone https://github.com/MondliWentandane/NoteTakingApp.git
cd NoteTakingApp
Install dependencies:
bash
npm install
Configure TypeScript (if needed):
bash
npm run build
Development
Available Scripts
Run the following commands from the project root:

Development: Start the development server
Build: Compile TypeScript to JavaScript
Test: Run test suite
Refer to package.json for the complete list of available scripts.

Key Features
Note Creation & Editing: Easily create and edit notes with a user-friendly interface
Organization: Organize notes using categories or tags
Context State Management: Leverages React Context for efficient state management
Type-Safe Code: Built with TypeScript for better code quality and developer experience
Cross-Platform: Runs on iOS and Android through React Native
Architecture
Components (/components)
Reusable UI components that compose the user interface.

Contexts (/contexts)
React Context API setup for global state management across the application.

Services (/services)
Business logic layer including data persistence, API calls, and core app logic.

Types (/types)
TypeScript interface and type definitions for improved type safety.

Utils (/utils)
Helper functions and utility modules used throughout the application.

Technologies Used
React Native - Mobile app framework
TypeScript - Type-safe JavaScript
React Context API - State management
npm - Package management
Configuration
TypeScript Configuration
TypeScript settings are defined in tsconfig.json. Modify this file to adjust compiler options.

React Native Configuration
App-specific configuration can be found in app.json, including app name, version, and plugins.
